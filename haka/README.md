# CSC Haka

Äänestyspalveluun voi kirjautua CSC Haka -federaatioon kuuluvasta
identiteetintarjoajasta (IdP).


## Tiivistetysti

- Kirjautuminen sallitaan ainoastaan esimerkiksi  Helsingin yliopiston
  käyttäjätunnuksille esikonfiguroimalla äänestyspalveluun halutun
  identiteetintarjoajan sertifikaatti ja palvelutunniste.

- Äänestäjä ohjataan suoraan esikonfiguroidulle identiteetintarjoajalle,
  jolloin  äänestäjä ei joudu kulkemaan CSC Hakan palveluvalinnan kautta.

- Äänestyspalvelu on rekisteröity
  [Hakan ylläpitämään resurssirekisteriin](https://rr.funet.fi/rr/).
